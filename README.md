# Secondo Market - Sneakers shop on React

[Try it on GitHub Pages](https://mikrocosmos.github.io/secondo-market/)

**Stack**:

- ReactJS
- hooks: useState, useEffect
- React Router
- Axios
- SCSS
- React Context
- React Skeleton

Based on course: React Sneakers by Archakov06.
